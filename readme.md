# baby-yoda

> A tiny (520B) layout animation module for react

- 🥚 **tiny**
- 🛸 **fast**: [FLIP animations](https://aerotwist.com/blog/flip-your-animations/) and Web Animations API
- 🧙‍♂️ **uses The Force**: magically animate layout changes like a Jedi
- 🐸 eats frogs
- 🍪 steals cookies
- 🤖 has a droid friend
- 👀 follows you
